# Prédiction de l'atterrissage du premier étage de la Falcon 9 de SpaceX\n
Dans ce travail, nous allons prédire si le premier étage de la fusée Falcon 9 atterrira avec succès. SpaceX annonce les lancements de la fusée Falcon 9 sur son site web, avec un coût de 62 millions de dollars.  si nous pouvons déterminer si le premier étage atterrira, nous pouvons déterminer le coût d'un lancement.
Nous allons donc partir de l'étape de la collecte des données jusqu'a la validation du modele.
